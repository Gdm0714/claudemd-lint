## Step 3: Apply Decisions

### If Accepted:
1. Find the target CLAUDE.md file (use Glob to search for `**/CLAUDE.md`)
2. Find or create the target section
3. Add the rule as a bullet point using the Edit tool
4. Update the suggestion's `status` to `"accepted"` in suggestions.json

### If Edited:
1. Let the user modify the rule text
2. Apply the modified rule to CLAUDE.md
3. Update the suggestion's `status` to `"accepted"` and store the final text

### If Dismissed:
1. Ask for a brief reason (optional)
2. Update the suggestion's `status` to `"dismissed"` and `dismissReason` in suggestions.json
